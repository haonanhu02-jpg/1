## 💡 About

YuanQue SCRM is a **100% open-source** private domain marketing solution based on WeCom (Enterprise WeChat). It provides a complete **"Acquisition → Management → Operations → Marketing → Service → Analytics"** pipeline for enterprise private domain marketing.

### Core Advantages

| Advantage | Description |
|-----------|-------------|
| 🤖 AI-Powered | Built on LangChain4j, supports multiple AI models with easy switching |
| 🚀 High Availability | Flexible and efficient deployment |
| 💰 Cost Effective | Reduce R&D costs while meeting customization needs |
| 🔒 Data Security | Full data control with independent IP ownership |
| 🔓 No Vendor Lock-in | Eliminate cloud dependency and vendor lock-in |
| ✅ Compliance | All features use official WeCom APIs for compliance |

---

## 📦 Open Source Features

### Core Features

- **Smart QR Codes** — Auto-tagging, auto-approve, duplicate prevention, custom welcome messages
- **Customer Acquisition Links** — External short links that auto-launch WeChat
- **Smart Group Codes** — Auto-create groups when full, auto-remark on join
- **Customer Pool** — Batch import leads and distribute evenly
- **Mass Messaging** — One-click mass send to customers or groups
- **Complaint Management** — Multi-channel notification + data analytics
- **Content Management** — Unified materials and scripts for consistent messaging
- **Customer Service** — Intelligent scheduling and routing

### AI Features

- **AI Assistant** — AI-powered marketing planning + intelligent navigation
- **AI Customer Service** — RAG-based knowledge base + auto-reply + human handoff
- **AI Chat Analysis** — Compliance review + intent detection for customers and groups
- **AI Insights** — Hot word analysis to discover customer interests
- **AI Smart Tags** — AI-generated customer and group tagging
