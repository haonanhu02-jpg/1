# 运营后台功能规划

## 1. 后台菜单结构

```text
运营工作台
渠道活码
员工池管理
欢迎语模板
素材库
客户标签
群池管理
分群规则
客户管理
进群 / 退群记录
统计报表
日志中心
告警中心
系统基础配置
```

---

## 2. 页面功能清单

| 页面 | 主要用途 | 运营动作 |
| --- | --- | --- |
| 运营工作台 | 查看核心数据和异常摘要 | 查看今日新增、进群、退群、告警 |
| 渠道活码 | 管理不同投放渠道的活码 | 新建、编辑、启用、停用、删除、查看二维码、复制、下载 |
| 员工池管理 | 管理承接客户的员工和员工池 | 新建员工池、编辑员工池、添加员工、移除员工、启用、停用 |
| 欢迎语模板 | 管理客户加好友后的欢迎内容 | 新建、编辑、启用、停用、预览、设置默认模板 |
| 素材库 | 管理欢迎语可引用的素材 | 新建文本、上传图片、上传文件、创建链接、启用、停用 |
| 客户标签 | 管理渠道标签、客户分层标签和群标签 | 新建、编辑、启用、停用、手动调整客户标签 |
| 群池管理 | 管理渠道对应的客户群集合 | 新建群池、编辑群池、绑定客户群、设置阈值、设置备用群、同步群信息 |
| 分群规则 | 配置客户进入哪个群池 | 新建、编辑、启用、停用、调整优先级、查看匹配记录 |
| 客户管理 | 查看客户档案和运营状态 | 查看客户、查看来源、查看标签、查看承接员工、手动调整标签 |
| 进群 / 退群记录 | 查看客户和客户群关系变化 | 查询、筛选、导出 |
| 统计报表 | 查看渠道、员工、群池转化数据 | 筛选、查看、导出 CSV / Excel |
| 日志中心 | 追踪完整业务链路 | 查询、筛选、查看详情 |
| 告警中心 | 处理业务异常 | 查看、筛选、标记已处理、记录处理备注 |
| 系统基础配置 | 配置默认欢迎语、默认群池等兜底项 | 查看、编辑、启用、停用 |

---

## 3. 关键业务字段

### 3.1 渠道活码

| 字段 | 说明 |
| --- | --- |
| 渠道名称 | 例如抖音、小红书、官网、线下展会 |
| 渠道编码 | 用于区分渠道来源 |
| 渠道标签 | 客户添加后自动打的标签 |
| 绑定员工池 | 客户扫码后可添加的员工范围 |
| 欢迎语模板 | 客户加好友后收到的欢迎语 |
| 默认群池 | 该渠道默认引导进入的群池 |
| 二维码 | 渠道投放使用的二维码 |
| 启用状态 | 启用、停用 |
| 创建时间 / 更新时间 | 管理追踪字段 |

### 3.2 员工池

| 字段 | 说明 |
| --- | --- |
| 员工 userid | 企业微信员工标识 |
| 员工名称 | 员工展示名称 |
| 手机号 / 备注 | 运营辅助识别 |
| 所属部门 | 员工组织归属 |
| 是否启用 | 停用员工不参与分流 |
| 员工池名称 | 例如销售一组、销售二组 |
| 员工池成员 | 员工池包含的员工 |

### 3.3 欢迎语模板

| 字段 | 说明 |
| --- | --- |
| 模板名称 | 运营识别名称 |
| 文本内容 | 欢迎语正文 |
| 客户昵称变量 | 用于个性化称呼 |
| 图片素材 | 可选 |
| 链接素材 | 可选 |
| 文件素材 | 可选 |
| 入群二维码 / 入群入口 | 引导客户进群 |
| 启用状态 | 启用、停用 |

### 3.4 素材库

| 字段 | 说明 |
| --- | --- |
| 素材名称 | 运营识别名称 |
| 素材类型 | 文本、图片、文件、链接 |
| 素材内容 | 文本或链接内容 |
| 素材 URL | 图片或文件地址 |
| 文件大小 | 文件类素材使用 |
| 启用状态 | 启用、停用 |
| 创建人 / 创建时间 | 管理追踪字段 |

### 3.5 客户标签

| 字段 | 说明 |
| --- | --- |
| 标签名称 | 例如抖音线索、已进群、VIP客户 |
| 标签类型 | 渠道标签、客户分层标签、群标签 |
| 启用状态 | 启用、停用 |
| 使用场景 | 自动打标、手动打标、分群规则 |

### 3.6 群池

| 字段 | 说明 |
| --- | --- |
| 群池名称 | 例如抖音客户群池 |
| 对应渠道 | 该群池默认服务的渠道 |
| 客户群 | 群池内的一个或多个企业微信客户群 |
| 当前人数 | 群当前人数 |
| 人数阈值 | 达到阈值后切换备用群 |
| 群状态 | 可用、满员、停用、异常 |
| 排序 | 决定备用群切换顺序 |
| 是否备用群 | 标识备用承接群 |
| 是否自动建群 | 一期作为可配置项确认，具体能力以后续技术方案为准 |

### 3.7 分群规则

| 字段 | 说明 |
| --- | --- |
| 规则名称 | 运营识别名称 |
| 渠道条件 | 按渠道匹配客户 |
| 标签条件 | 按客户标签匹配客户 |
| 目标群池 | 匹配后分配的群池 |
| 优先级 | 数字越高或越靠前越优先，需在产品规则中统一 |
| 启用状态 | 启用、停用 |

### 3.8 统计报表

| 字段 | 说明 |
| --- | --- |
| 时间范围 | 支持按时间筛选 |
| 渠道 | 支持按渠道筛选 |
| 员工 | 支持按承接员工筛选 |
| 群池 | 支持按群池筛选 |
| 新增好友数 | 客户添加员工成功数量 |
| 成功进群人数 | 客户进入客户群数量 |
| 退群人数 | 客户退出客户群数量 |
| 进群转化率 | 成功进群人数 / 新增好友数 |
| 导出 | CSV / Excel |

---

## 4. 页面验收要求

- 所有核心配置页应支持列表、详情、新建、编辑、启用、停用。
- 删除动作仅用于明确可删除的配置项；已有业务数据引用的配置应优先停用。
- 所有影响客户链路的配置变更需要留下操作记录。
- 统计、日志、告警类页面必须支持时间范围筛选。
- 导出内容应与当前筛选条件一致。
