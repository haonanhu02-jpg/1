# 统计指标清单

## 1. 一期核心指标

| 指标 | 口径 | 主要用途 |
| --- | --- | --- |
| 新增好友数 | 客户成功添加企业微信员工的数量 | 衡量渠道加好友效果 |
| 成功进群人数 | 客户成功进入客户群的数量 | 衡量入群转化效果 |
| 退群人数 | 客户退出客户群的数量 | 衡量群留存情况 |
| 进群转化率 | 成功进群人数 / 新增好友数 | 衡量从加好友到进群的转化 |
| 员工承接量 | 每个员工实际承接的客户数量 | 衡量员工承接压力 |
| 群当前人数 | 每个客户群当前人数 | 判断群容量状态 |
| 群池剩余容量 | 群池内可用客户群剩余承接空间 | 判断是否需要备用群 |
| 欢迎语发送成功数 | 欢迎语成功发送次数 | 衡量欢迎语链路正常率 |
| 欢迎语发送失败数 | 欢迎语发送失败次数 | 发现消息发送问题 |
| 欢迎语发送成功率 | 成功数 / 总发送数 | 评估欢迎语稳定性 |
| 接口异常数 | 外部能力调用失败次数 | 发现系统异常趋势 |

---

## 2. 筛选维度

| 维度 | 要求 |
| --- | --- |
| 时间范围 | 必须支持 |
| 渠道 | 必须支持 |
| 员工 | 必须支持 |
| 群池 | 必须支持 |
| 客户群 | 建议支持 |
| 标签 | 后续可扩展 |

---

## 3. 报表页面

### 3.1 渠道统计

展示每个渠道的：

- 新增好友数
- 成功进群人数
- 退群人数
- 进群转化率
- 欢迎语发送成功数
- 欢迎语发送失败数

### 3.2 员工统计

展示每个员工的：

- 承接客户数
- 客户来源渠道分布
- 客户进群人数
- 客户进群转化率

### 3.3 群池统计

展示每个群池的：

- 当前可用群数量
- 群当前人数
- 群池剩余容量
- 达到阈值的群数量
- 无可用群告警次数

### 3.4 异常统计

展示：

- 接口异常数
- 欢迎语发送失败数
- 标签打标失败数
- 群二维码失效数
- 回调异常数
- 数据同步失败数

---

## 4. 导出要求

- 统计报表支持导出 CSV / Excel。
- 导出内容应与当前筛选条件保持一致。
- 导出文件至少包含统计时间范围、筛选条件、指标名称、指标数值。
- 进群 / 退群记录支持导出明细。
- 告警列表支持导出明细。

---

## 5. 扫码量统计边界

一期默认不承诺精准统计“只扫码但未添加好友”的用户行为。

原因：

- 如果直接投放企业微信联系我二维码，系统通常只能在客户添加员工成功后拿到后续事件。
- 只扫码但未添加好友的行为可能无法完整记录。

如需精准扫码量统计，需要另做扩展方案：

```text
客户扫描我们自己的统计二维码
↓
系统记录访问
↓
页面展示或跳转企业微信联系我二维码
```

该能力不属于一期默认范围。
